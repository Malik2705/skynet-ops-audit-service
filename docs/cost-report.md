# Cost Report

If deployed on AWS ECS Fargate:

Estimated monthly cost:

ECS Fargate (0.25 vCPU, 0.5GB RAM)
~ $10 – $15 / month

CloudWatch Logs
~ $1 – $2 / month

Total estimated cost:
~ $12 – $17 per month

Cost optimization:
- Use smaller container size
- Enable log retention limits
- Use autoscaling
